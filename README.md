# UpSkill - Skill Exchange Platform

A full-stack web application that enables users to exchange skills using a credit-based system.

## 🚀 Features
- User authentication
- Profile management
- Skills offered & wanted
- Skill request system
- Credit-based exchange:
  - Request → credits deducted
  - Accept → teacher earns credits
  - Reject → credits refunded

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: MongoDB

## ⚡ Status
This is an MVP (Minimum Viable Product). Backend currently runs locally.

## 💡 Future Improvements
- Session-based authentication
- Deployment
- Notifications system
- Rating system
